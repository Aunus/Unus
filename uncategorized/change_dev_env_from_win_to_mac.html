        Zoe 2018/5/5 Mac

        # 记Windows开发迁移至Mac环境

        第一次使用Mac pro，感觉整个迁移过程还是令人心酸的。基本耗时很久，但好的事是大部分都装好了，补一下安装过程。

        本人Java开发一枚，偶尔用Python和Node.js。最近工作环境从windows迁移到了Mac os, 自己给自己挖了不少坑。Mac的基本操作就不说了，官方教程都有，很容易上手，说一下开发环境配置和常用软件。

        # Java 多版本配置

        当前常用的jdk7、jdk8，先从官网下载Mac OS X x64 版本，Java版本归档地址：Oracle Java Archive 。下载后默认安装至/Library/Java/JavaVirtualMachines

        多版本切换问题，同Linux和windows一样，需要配置环境变量。

        补充一下: Mac 也分用户个人级别环境变量和系统级别环境变量。/etc/profile和/etc/paths是系统级,~/.bash_profile ~/.bash_login ~/.profile ~/.bashrc 用户级。加载次序依次为:

        /etc/profile /etc/paths ~/.bash_profile ~/.bash_login ~/.profile ~/.bashrc

        因为懒直接配置到了/etc/profile 上面。在末尾追加：

        export JAVA_7_HOME=/Library/Java/JavaVirtualMachines/jdk1.7.0_80.jdk/Contents/Home
export JAVA_8_HOME=/Library/Java/JavaVirtualMachines/jdk1.8.0_171.jdk/Contents/Home
export JAVA_HOME=$JAVA_7_HOME

alias jdk8='export JAVA_HOME=$JAVA_8_HOME'

alias jdk7='export JAVA_HOME=$JAVA_7_HOME'

        直接在bash上输入jdk7/jdk8可以直接修改当前jdk版本

        • Jdk7
        xx@MacBook-Pro ~$ jdk7 
xx@MacBook-Pro ~$ java -version

        # Tomcat配置

        直接从Tomcat 官网 下载的版本，zip/tar.gz格式，下载后解压安装到/Library/Tomcat/apache-tomcat-7.0.86。

        IntelliJ IDEA使用Tomcat报错：

        error=13, permission denied

        需要设置权限

        cd /Library/Tomcat/apache-tomcat-7.0.86/bin

chmod 777 *.sh

        # Shuttle(已经放弃)

        Shuttle 避免ssh远程连接频繁输入host、port。开源免费：http://fitztrev.github.io/shuttle/，但是需输入password.

        懒人还需安装**sshpass **，借鉴stackoverflow的问题，通过brew安装。

        先安装brew, brew官网：https://brew.sh/index_zh-cn.html

        后安装sshpass

        brew install https://raw.githubusercontent.com/kadwanev/bigboybrew/master/Library/Formula/sshpass.rb

        # 其他开发软件

        安装时.app结尾程序需要拖到应用程序文件夹内,部分安装时会提示将程序放进去

        # IntelliJ IDEA 配置

        jetbrains 官网 下载IDEA，默认安装配置。常用使用同Windows下，除快捷键不同，无太大差别。

        # DataGrip(已经放弃)

        jetbrains旗下数据库客户端，idea付费版有插件支持datagrip全部功能，不过单独装一个datagrip比较合适，避免多项目的时候切换比较麻烦。配置不复杂，搜一下即可。

        2019.06.01补充：复制表的时候结构不会全量复制，需要每个字段逐一更改，修改字段及创建新的表的时候也很麻烦，已经放弃使用，改为Navicate Premium

        # Navicate Premium

        windows用习惯了，还行。

        # SVN

        mac 自带svn，前提需要安装Xcode。 svn客户端使用的是Cornerstone，收费。先配置repository ，后check out。

        # Git

        git 官网 下载mac版本，看官方自带document就好。Git 客户端用Sourcetree，开源免费。

        git推荐使用ssh连接，先在本地生成ssh 私钥和公钥，再将.ssh目录下的id_rsa.pub公钥传到github上配置，私钥请保密。

        $ssh-keygen -t rsa -C "youremail@example.com"

        # VSCode

        微软开源编辑器，官网下载就可以。

        # Markdown编辑器

        vscode支持markdown，idea也支持，另外推荐Typora ，开源，简洁，美观，前提需熟悉markdown语法。官网：https://www.typora.io/

        # VPN

        Mac os 12.13抛弃了pptp协议，pptp协议需要安装Shimo ，收费软件。

        SS 则使用ShadowsocksX-NG ，开源免费。下载地址：https://github.com/shadowsocks/ShadowsocksX-NG/releases/

        # Docker

        开发环境搭建利器，docker-compose.yml配置完后部署十分舒服。
